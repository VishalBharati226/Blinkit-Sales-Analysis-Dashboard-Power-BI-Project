📊 Blinkit Sales Analysis Dashboard | Power BI Project
🔍 Project Overview

This project presents an interactive Power BI dashboard built to analyze Blinkit’s retail sales performance across multiple dimensions such as outlet type, outlet size, location tier, item category, fat content, and establishment year.
The dashboard helps stakeholders quickly identify sales trends, performance drivers, and business insights.

🎯 Objectives

Analyze total and average sales performance

Compare outlet performance by size, type, and location

Identify top-performing item categories

Understand customer preference through ratings and item visibility

Track sales trends over outlet establishment years

📌 Key KPIs

Total Sales: $1.20M

Average Sales: $141

Number of Items: 8,523

Average Rating: 3.9

📈 Dashboard Features

Sales Trend Analysis: Sales by outlet establishment year

Category Analysis: Item-wise sales distribution

Outlet Performance:

Outlet Size (Small, Medium, High)

Outlet Location (Tier 1, Tier 2, Tier 3)
